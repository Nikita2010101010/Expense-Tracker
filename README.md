# Expense-Tracker
Никита Горлов Описание программы,это приложение для отслеживания расходов.Введите сумму,выберите категорию,укажите дату,нажмите добавить расход,для фильтрации выберите категорию и дату,чтобы увидеть подходящие записи Пример:добавление расхода:сумма 500,категория Еда,дата сегодня,фильтрация по категории транспорт,подсчет общей суммы расходов
